\newglossaryentry{Same-Origin-Policy}{
	name={Same-Origin-Policy},
	description={%
		Regeln für den Browser, nach denen eine Webanwendung nicht auf die Daten einer fremden Webanwendung zugreifen kann. Im Standardfall hat jeder Code genau einen Sichtbarkeitsbereich mit lokalen Daten, wenn dieser über genau das gleiche Protocol, den gleichen Host und den gleichen Port vom Server geliefert wird. Es gibt etablierte Techniken diese Policy zu umgehen, bei denen sich der Code im Browser und der Server eindeutig identifizieren
	}
}
