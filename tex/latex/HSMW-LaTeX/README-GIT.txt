Mitwirkung an der Weiterentwicklung von HSMW-LaTeX
==================================================

Falls Sie an der Weiterentwicklung von HSMW-LaTeX mitwirken wollen, so
schicken Sie mir bitte Ihren öffentlichen SSH-Schlüssel als PGP-signierte
E-Mail an dohmen@hs-mittweida.de.

Nachdem ich dessen Empfang bestätigt habe, wechseln Sie bitte ins
HSMW-LaTeX-Verzeichnis und geben Folgendes ein:

git remote set-url origin git@hsmw.ddns.net:HSMW-LaTeX.git

Danach haben Sie schreibenden Zugriff auf das Git-Repository. 

Für Fragen und Anregungen stehe ich gern zur Verfügung.

K. Dohmen
